# analysis Modul

Beinhaltet Scoring-Logik, KI-Prompting und Report-Generator.

**Dateien:**
- `finbert.py`: Anbindung an den lokalen FinBERT-Service
- `deepseek.py`: Aufruf für Extraktion und Zusammenfassung (DeepSeek API)
- `kimi.py`: Aufruf für detaillierte Call-Analyse
- `scoring.py`: Opportunity- und Torpedo-Score Berechnung
- `report_generator.py`: Baut die Markdown/HTML Reports und Makro-Header
